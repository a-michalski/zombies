# Construction Spot Sprite

## Instrukcja

1. **Wygeneruj grafikę** używając promptu z `docs/ai-prompts-missing-elements.md` (sekcja 1)
2. **Wrzuć tutaj** wygenerowany plik PNG (może mieć dowolną nazwę)
3. **Uruchom skrypt** do przetworzenia:
   ```bash
   node scripts/process-construction-spot.js
   ```

## Wymagania

- **Rozmiar docelowy**: 32x32px
- **Format**: PNG z przezroczystością
- **Opis**: Sprite miejsca budowy wieży - podświetlony punkt/platforma z pomarańczowo-złotymi akcentami

## Plik docelowy

Po przetworzeniu plik zostanie zapisany jako:
`assets/images/map/construction-spot.png`
