# End Waypoint - Wrak Statku

## Instrukcja

1. **Wygeneruj grafikę** używając promptu z `docs/ai-prompts-missing-elements.md` (sekcja 4)
2. **Wrzuć tutaj** wygenerowany plik PNG (może mieć dowolną nazwę)
3. **Uruchom skrypt** do przetworzenia:
   ```bash
   node scripts/process-end-waypoint.js
   ```

## Wymagania

- **Rozmiar docelowy**: 48x48px (lub większy dla efektu)
- **Format**: PNG z przezroczystością
- **Opis**: Niebieski wrak statku z energią, do którego wchodzą zombie
- **Kolory**: Niebieskie światło (#4444FF, #5C9EFF, #1E3A8A)

## Plik docelowy

Po przetworzeniu plik zostanie zapisany jako:
`assets/images/map/end-waypoint.png`
