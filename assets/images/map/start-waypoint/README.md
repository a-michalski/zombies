# Start Waypoint - Czeluść

## Instrukcja

1. **Wygeneruj grafikę** używając promptu z `docs/ai-prompts-missing-elements.md` (sekcja 3)
2. **Wrzuć tutaj** wygenerowany plik PNG (może mieć dowolną nazwę)
3. **Uruchom skrypt** do przetworzenia:
   ```bash
   node scripts/process-start-waypoint.js
   ```

## Wymagania

- **Rozmiar docelowy**: 48x48px (lub większy dla efektu)
- **Format**: PNG z przezroczystością
- **Opis**: Czerwony otwór/czeluść z ogniem, z którego wychodzą zombie
- **Kolory**: Czerwono-pomarańczowe światło (#FF4444, #FF6B6B, #8B0000)

## Plik docelowy

Po przetworzeniu plik zostanie zapisany jako:
`assets/images/map/start-waypoint.png`
