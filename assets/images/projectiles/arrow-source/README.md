# Arrow Projectile - Poprawiona wersja

## Instrukcja

1. **Wygeneruj grafikę** używając promptu z `docs/ai-prompts-missing-elements.md` (sekcja 2)
2. **Wrzuć tutaj** wygenerowany plik PNG (może mieć dowolną nazwę)
3. **Uruchom skrypt** do przetworzenia:
   ```bash
   node scripts/process-arrow.js
   ```

## Wymagania

- **Rozmiar docelowy**: 16x32px
- **Format**: PNG z przezroczystością
- **Opis**: Strzała z kuszy - brązowy trzonek (#8B4513), metalowy grot (#C0C0C0)
- **Ważne**: Strzała powinna być widoczna (nie biała!), z wyraźnymi kolorami

## Plik docelowy

Po przetworzeniu plik zostanie zapisany jako:
`assets/images/projectiles/arrow.png` (nadpisze istniejący)
