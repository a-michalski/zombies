# Level 3 Tower Images

Wrzuć tutaj obraz wieży poziomu 3.

**Wymagania:**
- Format: PNG, JPG lub JPEG
- Rozmiar: Dowolny (zostanie automatycznie przeskalowany do 64x64px)
- Nazwa: Dowolna (pierwszy znaleziony obraz zostanie użyty)

**Jak użyć:**
1. Wrzuć obraz do tego folderu
2. Uruchom: `node scripts/process-tower-levels.js`
3. Obraz zostanie przetworzony i zapisany jako `lookout-post-level-3.png`
