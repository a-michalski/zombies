# Runner Enemy Images

Wrzuć tutaj obraz zombie typu "runner" (szybki wrog).

**Wymagania:**
- Format: PNG, JPG lub JPEG
- Rozmiar: Dowolny (zostanie automatycznie przeskalowany do 44x44px)
- Nazwa: Dowolna (pierwszy znaleziony obraz zostanie użyty)

**Jak użyć:**
1. Wrzuć obraz do tego folderu
2. Uruchom: `node scripts/process-enemy-images.js`
3. Obraz zostanie przetworzony i zapisany jako `runner.png`
