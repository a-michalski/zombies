# Brute Enemy Images

Wrzuć tutaj obraz zombie typu "brute" (silny, wytrzymały wrog).

**Wymagania:**
- Format: PNG, JPG lub JPEG
- Rozmiar: Dowolny (zostanie automatycznie przeskalowany do 64x64px)
- Nazwa: Dowolna (pierwszy znaleziony obraz zostanie użyty)

**Jak użyć:**
1. Wrzuć obraz do tego folderu
2. Uruchom: `node scripts/process-enemy-images.js`
3. Obraz zostanie przetworzony i zapisany jako `brute.png`
