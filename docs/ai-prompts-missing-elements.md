## Uwagi techniczne:

- **Wszystkie grafiki**: top-down view (widok z góry), PNG z przezroczystym tłem (oprócz path-texture i background)
- **Kolory tła mapy**: JAŚNIEJSZE niż UI elementy - użyj stonowanych zimowych kolorów (jasne szarości, beże, brązy) dla lepszej czytelności
- **Kolory innych elementów**: zgodne z paletą gry (ciemne, post-apokaliptyczne)
- **Styl**: realistyczny, nie kreskówkowy, wysokiej jakości
- **Spójność**: wszystkie elementy w jednym stylu artystycznym
- **Scenografia tła**: las (prawa) → polana (środek) → plaża z wrakiem (lewa)
- **Gotowe do użycia**: każdy prompt można skopiować i wkleić bezpośrednio do AI
