
### Dla DALL-E 3:
```
Game sprite, pixel art style or high detail 2D illustration, transparent background, centered composition
```

### Dla Stable Diffusion:
```
game sprite, top-down view, post-apocalyptic, tower defense, detailed, 2D illustration, transparent background, centered, game asset quality
```

## Wspólne elementy wszystkich promptów:

1. **Perspektywa**: "Top-down view" lub "Isometric 3/4 view"
2. **Styl**: "Post-apocalyptic", "Realistic detailed", "Game sprite"
3. **Kolory**: Konkretne hex codes gdzie wymagane
4. **Rozmiar**: Dokładne wymiary w pikselach
5. **Format**: "PNG transparent background", "Centered composition"
6. **Jakość**: "Game asset quality", "High detail", "Realistic"

---

## Przykład pełnego promptu (Lookout Post Level 1):

```
Top-down view game sprite of a small lookout post tower, post-apocalyptic tower defense game style. Metal defensive structure on a platform, crossbow weapon mounted on top, small survivor figure visible. Realistic detailed, industrial metal with rust, battle damage, dark shadows. Color scheme: dark gray metal base (#2a2a2a), blue metal accents (#4A90E2), rust brown details. 64x64 pixels, PNG transparent background, centered composition, game asset quality, high detail 2D illustration, top-down perspective, post-apocalyptic style
```

## Uwagi:

- Wszystkie grafiki powinny być w **top-down view** (widok z góry)
- Kolory powinny być zgodne z obecną paletą gry (ciemne, post-apokaliptyczne)
- Wszystkie elementy powinny mieć **przezroczyste tło** (PNG z alpha)
- Spójny styl artystyczny w całej grze
- Realistyczny, nie kreskówkowy
- Wysokiej jakości, szczegółowe
